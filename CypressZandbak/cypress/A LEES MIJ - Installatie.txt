Installatie

----------------------------------------------------------------------------------

Als je de EERSTEMAAND hebt gecloned moet je nog 
- Node.js en 
- Cypress 
installeren.

1. Download node.js: https://nodejs.org/en/download
je kiest hier de windows installer 64 bit. Na het downloaden installeer je de software. 
Gewoon overal op next klikken. 

2. Download Git: https://git-scm.com/downloads
je kiest hier de windows installer 64 bit. Na het downloaden installeer je de software. 
Gewoon overal op next klikken. 

3. Na installatie open je VSCode. In de Explorer van VSCode ga je met de cursor op de 
map CypressZandbak staan. Klik nu op de rechtermuisknop. Er verschijnt nu een pop-up menu.
In dat pop-up menu kies je: 'Open in Integrated Terminal'. 

In de terminal rechtsbeneden in het scherm moet je vervolgens de instructies plakken 
die hieronder staan. 

-- Commando's voor de Terminal --
Voor de koppeling van Node.js met de CypressZandbak plak je dit in de terminal:
npm install --no-save

Daarna plak je dit in de terminal:
npm update --no-save

En daarna dit voor de installatie van Cypress:
npm install cypress@10.11.0

Je bent nu klaar om Cypress te laten runnen. 

Als dit gelukt is ben je klaar om de boel op te starten met dit commando: 
npx cypress open auto-start --e2e --browser chrome
