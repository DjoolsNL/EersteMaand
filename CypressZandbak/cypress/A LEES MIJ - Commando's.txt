
Als je de Zandbak hebt gecloned zul je nog node.js en cypress moeten installeren.
Daarvoor moet je de terminal openen. Die vind je in het menu View. 


-- Commando's voor de Terminal --
xx installatie Cypress:
npm install --no-save
npm update --no-save
npm install cypress@10.11.0

xx start Cypress applicatie: 
npx cypress open auto-start --e2e --browser chrome
