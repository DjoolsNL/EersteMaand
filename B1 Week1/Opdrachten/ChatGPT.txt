Opdracht - ToolSkills omgaan met ChatGPT

De OPDRACHT
Vraag ChatGPT:
- wat kan ik je allemaal vragen?
- waar ben je goed in?
- hoe kan je me helpen?
- wat zijn slimme vragen waarmee ik het beste uit je kan halen?
- met welke instructies die ik je geven kan krijg ik de beste resultaten?
- als je zelf een beetje een gok moet maken wat het meest passende antwoord
is op een vraag van mij, probeer dan niet iets te antwoorden dat een gok is
maar vraag me liever de instructie te verduidelijken. 

WAAROM
ChatGPT is je persoonlijke leraar die 24 uur per dag klaar staat om je
vragen te beantwoorden. Hij weet heel veel van allerlei programmeertalen
en van IT. Je kunt hem code 'voeren' en hij legt je uit wat die code precies doet.
Je kunt hem ook vragen code te schrijven maar dan moet je wel specifiek omschrijven
wat die code moet doen.
